========================
FEATURE 1: FOLLOW SYSTEM
========================

Users should be able to follow/unfollow other users.

Database:
Create table:
- follows
  - id (PK)
  - follower_id (FK users.id)
  - following_id (FK users.id)
  - created_at

Backend:
- POST /api/users/:id/follow
- DELETE /api/users/:id/follow
- GET /api/users/:id/followers
- GET /api/users/:id/following

Rules:
- Cannot follow self
- Prevent duplicate follow
- Return follower count

Frontend:
- Add "Follow" button on user profile & comment username hover
- Toggle button (Follow / Following)
- Show follower count

FEATURE 3: XP + LEVEL SYSTEM
========================

Gamification system:

Database (update users table):
- xp (default 0)
- level (default 1)
- watch_time (seconds)

XP Rules:
- Comment = +5 XP
- Reply = +3 XP
- Watching 10 min = +10 XP

Level Formula:
- Level = floor(xp / 100)

Backend:
- Automatically update XP on actions
- Return updated level

Frontend:
- Show level badge near username
- Show XP progress bar in profile
